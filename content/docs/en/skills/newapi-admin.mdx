## Introduction

**gravitexapi-admin** is a Skill plugin designed for Gravitex API administrators. Unlike the user-level [gravitexapi](/en/docs/skills/gravitexapi) Skill, gravitexapi-admin focuses on **admin operations**, allowing site administrators to perform daily management tasks from within their AI editor using natural language commands — no need to log into the admin dashboard.

## Planned Features

### Channel Management

| Command (Planned) | Description |
|-------------------|-------------|
| `/gravitexapi-admin channels` | List all channels and their status |
| `/gravitexapi-admin channel-add` | Add new channels (OpenAI, Claude, Gemini, Azure, etc.) |
| `/gravitexapi-admin channel-test <id>` | Test connectivity of a specific channel |
| `/gravitexapi-admin channel-toggle <id>` | Enable or disable a specific channel |
| `/gravitexapi-admin channel-priority` | View and adjust channel priority |

### User Management

| Command (Planned) | Description |
|-------------------|-------------|
| `/gravitexapi-admin users` | List users |
| `/gravitexapi-admin user-info <id>` | View user details (balance, group, usage) |
| `/gravitexapi-admin user-quota <id>` | Adjust user quota and balance |
| `/gravitexapi-admin user-group <id> <group>` | Change user's group |
| `/gravitexapi-admin user-ban <id>` | Ban or unban a user |

### System Configuration

| Command (Planned) | Description |
|-------------------|-------------|
| `/gravitexapi-admin config` | View current system configuration |
| `/gravitexapi-admin config-set <key> <value>` | Modify a system configuration item |
| `/gravitexapi-admin ratio` | View model rate multiplier settings |
| `/gravitexapi-admin ratio-sync` | Sync upstream model rate multipliers |

### Logs & Monitoring

| Command (Planned) | Description |
|-------------------|-------------|
| `/gravitexapi-admin logs` | View recent request logs |
| `/gravitexapi-admin stats` | View system statistics (request volume, consumption, error rate) |
| `/gravitexapi-admin status` | Check system health and channel connectivity |

### Token & Redemption Management

| Command (Planned) | Description |
|-------------------|-------------|
| `/gravitexapi-admin redemptions` | List redemption codes |
| `/gravitexapi-admin redemption-create` | Batch create redemption codes |
| `/gravitexapi-admin token-overview` | Global token usage overview |

## Comparison with gravitexapi Skill

| | gravitexapi | gravitexapi-admin |
|---|--------|-------------|
| **Target Users** | Regular users / developers | Site administrators |
| **Scope** | Personal models, tokens, balance, groups | Global channels, users, config, logs |
| **Permissions** | User access token | Admin access token |
| **Status** | Released | Under development |
