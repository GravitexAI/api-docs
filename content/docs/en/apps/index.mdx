## Integration Guide

Most applications only require configuring the following information to connect to Gravitex API:

1.  **API Address**: Your Gravitex API service address
2.  **API Key**: The Token generated in the Gravitex API console
3.  **Model Name**: Select the model to use

For specific configuration steps, please refer to the detailed documentation of each application.
