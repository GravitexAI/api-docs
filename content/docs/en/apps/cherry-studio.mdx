<Callout type="info" title="Chat Settings Options">
  In the System Settings -> Chat Settings of the Gravitex API console, the following
  shortcut options can be added to easily populate Cherry Studio with a single
  click on the token management page:
</Callout>

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

## Gravitex API Integration Method

### Parameter Configuration

Provider Type: Gravitex API supported types
API Key: Obtained from Gravitex API
API Address: Gravitex API site address

### Steps

1. Copy your API Key from Gravitex API
{/* ![Copy API Key](/assets/cherry_studio/copy_api_key.png) */}
2. Add a provider in Cherry Studio, fill in the API address and key
{/* ![Add Provider](/assets/cherry_studio/add_provider.png) */}
3. Add the models you want to use
{/* ![Add Models](/assets/cherry_studio/add_models.png) */}
4. Return to the chat page and switch to a Gravitex API model to start chatting
{/* ![Switch to Chat Page](/assets/cherry_studio/back_to_chat.png) */}
{/* ![Switch Model](/assets/cherry_studio/switch_model.png) */}
