Seedance 2.0 通过 Gravitex AI 网关提供统一视频生成能力，支持文生视频、图生视频（首帧/首尾帧）、多模态参考（图/视频/音频）及素材库 `asset://` 面部一致性引用。完整场景说明与代码示例见仓库内《Seedance 2.0.md》用户文档。

**模型**：`seedance-2-0`（480p / 720p / 1080p）、`seedance-2-0-fast`（480p / 720p，不支持 1080p）

## 视频生成

- [创建视频生成任务](/zh/docs/api/ai-model/videos/seedance2.0/createvideogeneration) — `POST /v1/video/generations`
- [查询任务状态](/zh/docs/api/ai-model/videos/seedance2.0/getvideogeneration) — `GET /v1/video/generations/{task_id}`（建议每 5 秒轮询）
- [查询审核拦截原因](/zh/docs/api/ai-model/videos/seedance2.0/getmoderationresult) — `GET /v1/video/generations/moderation-result/{task_id}`（仅 failed 任务，需白名单）

## 素材库

素材库接口均使用 `Content-Type: application/json`，**仅接受公网 https URL**（不支持 Base64 / multipart 直传）。虚拟组（`aigc`）与真人组（`liveness_face`）均支持 Image / Video / Audio。

- [创建素材组](/zh/docs/api/ai-model/videos/seedance2.0/createassetgroup) — `POST /v1/asset-groups`（虚拟 `aigc`）
- [列出素材组](/zh/docs/api/ai-model/videos/seedance2.0/listassetgroups) — `GET /v1/asset-groups`（默认 `group_type=aigc`）
- [删除素材组](/zh/docs/api/ai-model/videos/seedance2.0/deleteassetgroup) — `DELETE /v1/asset-groups/{group_id}`
- [创建素材](/zh/docs/api/ai-model/videos/seedance2.0/createasset) — `POST /v1/assets`
- [列出素材](/zh/docs/api/ai-model/videos/seedance2.0/listassets) — `GET /v1/assets`（默认 `group_type=aigc`）
- [查询单个素材](/zh/docs/api/ai-model/videos/seedance2.0/getasset) — `GET /v1/assets/{virtual_id}`
- [删除素材](/zh/docs/api/ai-model/videos/seedance2.0/deleteasset) — `DELETE /v1/assets/{virtual_id}`

## 真人素材组（H5 活体核验）

- [发起核验会话](/zh/docs/api/ai-model/videos/seedance2.0/createvisualvalidatesession) — `POST /v1/visual-validate/session`
- [核验结果回调（内部）](/zh/docs/api/ai-model/videos/seedance2.0/visualvalidateresult) — `POST /v1/visual-validate/result`（网关回调页调用，业务侧通常无需直接请求）
