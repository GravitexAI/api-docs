<Callout type="info" title="チャット設定オプション">
  Gravitex API
  コンソールのシステム設定->チャット設定で、以下のショートカットオプションを追加できます。これにより、トークン管理ページからCherry
  Studioへワンクリックで入力するのに便利です：
</Callout>

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

<Callout type="info">
  🍒 Cherry
  Studioは、プロフェッショナルユーザー向けに設計された機能豊富なデスクトップAIクライアントです。30以上の業界インテリジェントアシスタントを統合しており、様々な作業シナリオのニーズを満たし、作業効率を大幅に向上させます。
</Callout>

## Gravitex API 接続方法

### パラメータ入力

プロバイダータイプ：Gravitex APIがサポートするタイプ  
APIキー：Gravitex APIで取得  
APIアドレス：Gravitex APIサイトアドレス

### 操作手順

1. Gravitex APIでAPIキーをコピー
{/* ![APIキーをコピー](/assets/cherry_studio/copy_api_key.png) */}
2. Cherry Studioでプロバイダーを追加し、APIアドレスとキーを入力
{/* ![プロバイダーを追加](/assets/cherry_studio/add_provider.png) */}
3. 使用したいモデルを追加
{/* ![モデルを追加](/assets/cherry_studio/add_models.png) */}
4. チャットページに戻り、Gravitex APIモデルに切り替えて会話を開始
{/* ![チャットページに切り替える](/assets/cherry_studio/back_to_chat.png) */}
{/* ![モデルを切り替える](/assets/cherry_studio/switch_model.png) */}
