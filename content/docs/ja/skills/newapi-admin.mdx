## はじめに

**gravitexapi-admin** は Gravitex API 管理者向けの Skill プラグインです。ユーザー向けの [gravitexapi](/ja/docs/skills/gravitexapi) Skill とは異なり、gravitexapi-admin は**管理者運用シナリオ**に特化しており、サイト管理者が AI エディター内で自然言語コマンドを使って日常の管理タスクを実行できます。管理ダッシュボードにログインする必要はありません。

## 計画中の機能

### チャネル管理

| コマンド（計画中） | 説明 |
|-------------------|------|
| `/gravitexapi-admin channels` | 全チャネルとそのステータスを一覧表示 |
| `/gravitexapi-admin channel-add` | 新しいチャネルを追加（OpenAI、Claude、Gemini、Azure など） |
| `/gravitexapi-admin channel-test <id>` | 指定チャネルの接続性をテスト |
| `/gravitexapi-admin channel-toggle <id>` | 指定チャネルの有効化・無効化 |
| `/gravitexapi-admin channel-priority` | チャネル優先度の確認と調整 |

### ユーザー管理

| コマンド（計画中） | 説明 |
|-------------------|------|
| `/gravitexapi-admin users` | ユーザー一覧を表示 |
| `/gravitexapi-admin user-info <id>` | ユーザー詳細を確認（残高、グループ、使用量） |
| `/gravitexapi-admin user-quota <id>` | ユーザーのクォータと残高を調整 |
| `/gravitexapi-admin user-group <id> <group>` | ユーザーのグループを変更 |
| `/gravitexapi-admin user-ban <id>` | ユーザーのBAN・BAN解除 |

### システム設定

| コマンド（計画中） | 説明 |
|-------------------|------|
| `/gravitexapi-admin config` | 現在のシステム設定を確認 |
| `/gravitexapi-admin config-set <key> <value>` | システム設定項目を変更 |
| `/gravitexapi-admin ratio` | モデル倍率設定を確認 |
| `/gravitexapi-admin ratio-sync` | 上流のモデル倍率を同期 |

### ログとモニタリング

| コマンド（計画中） | 説明 |
|-------------------|------|
| `/gravitexapi-admin logs` | 最近のリクエストログを確認 |
| `/gravitexapi-admin stats` | システム統計を確認（リクエスト量、消費、エラー率） |
| `/gravitexapi-admin status` | システムの健全性とチャネル接続性を確認 |

### トークンと引き換えコード管理

| コマンド（計画中） | 説明 |
|-------------------|------|
| `/gravitexapi-admin redemptions` | 引き換えコードを一覧表示 |
| `/gravitexapi-admin redemption-create` | 引き換えコードを一括作成 |
| `/gravitexapi-admin token-overview` | グローバルトークン使用概要 |

## gravitexapi Skill との違い

| | gravitexapi | gravitexapi-admin |
|---|--------|-------------|
| **対象ユーザー** | 一般ユーザー / 開発者 | サイト管理者 |
| **操作範囲** | 個人のモデル、トークン、残高、グループ | グローバルチャネル、ユーザー、設定、ログ |
| **必要な権限** | ユーザーアクセストークン | 管理者アクセストークン |
| **ステータス** | リリース済み | 開発中 |
